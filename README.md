# Plant shop UI Design - Figma
This project showcases a visually appealing UI design for a plant website, crafted using Figma and Photoshop. The design focuses on creating a user-friendly interface that highlights the beauty and variety of plants while ensuring seamless navigation. From wireframing to the final visual design, this project reflects creativity and functionality, making it an ideal resource for plant enthusiasts and businesses.
